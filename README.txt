1. Run index.html on localhost
2. Control Options
(1) Ambient light color
(2) Model
        (a) Shading - Flat, Gouraud, Phong, Toon
        (b) Material - Ambient, Diffuse, Specular reflection coefficient, Shininess
        (c) Position
        (d) Rotation
        (e) Scale
        (f) Shear
        (g) Texture(Only for Teapot)
(3) Light Source
        (a) Position
        (b) Color - Diffuse, Specular
